「Minecraft」OP 权限等级

2018-07-02
游戏
Minecraft · 科普

可在 Minecraft 官方服务端的 ops.json 文件内进行修改。

等级

  1. 无视重生点保护
  2. 执行 /clear /difficulty /effect /gamemode /gamerule /give /tp 命令，编辑命令方块
  3. 执行 /ban /deop /kick /op 命令
  4. 执行 /stop 命令
