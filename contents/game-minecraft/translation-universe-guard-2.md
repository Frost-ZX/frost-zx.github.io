# 「Minecraft」UniverseGuard 2 插件部分提示文本汉化

服务端平台：Sponge API 7.1.5
插件版本：2.17

## 使用方法

使用文本编辑工具（例如 Visual Studio Code），复制相应字段到本插件的配置文件（universeguard.conf）中替换相应的字段。
